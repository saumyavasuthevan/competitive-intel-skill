# competitive-intel-skill
Automated competitive intelligence reports for Product Managers

#### What's Included
- Automated competitive research via Brave Search
- HTML email reports with TL;DR and detailed analysis
- Weekly scheduling support
- Interactive setup wizard

#### Set Up
Upon installation, the wizard will interactively ask for:
* **Target Company** - The company to analyze
* **Recipient Email** - Where to send reports
* **Sender Gmail** - Which Gmail account sends the reports
* **Schedule** - Weekly, daily, or custom
* **Timezone** - Your local timezone
